# 成舍企業 合約產生系統

靜態網頁應用,用於產生工程合約、設計合約、客變專用合約三種合約書。

## 使用方式

直接用瀏覽器開啟 `index.html` 即可使用,不需要伺服器或安裝套件。

## 功能

- 三種合約類型切換(工程合約 / 設計合約 / 客變專用合約)
- 表單填寫甲方資料、金額、日期,即時預覽整份合約
- 依經辦分公司自動帶入乙方資訊(可手動修改)
- 金額自動換算國字大寫、西元日期自動轉民國年
- 工程合約付款方案可選預設(小型商辦 / 一般案件・大型商辦)或自訂分期
- 設計合約付款階段比例可調整
- 列印 / 另存 PDF(瀏覽器內建列印功能)
- 另存 Word(.docx)

## 檔案結構

- `index.html` - 主頁面與表單
- `styles.css` - 樣式
- `templates.js` - 合約範本文字
- `app.js` - 互動邏輯與資料
- `assets/logo.png` - 公司 LOGO
- `lib/html-docx.js` - Word 匯出用第三方函式庫
